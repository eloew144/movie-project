Movie Database

This program is a list of 6 movies that I enjoy that are shown through an HTML file. The movies will show their title, a poster picture, and a trailer when the poster picture is clicked.

Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

Prerequisites

An understanding of how to run Python code.


Installing

Save code in one file. 
Run the HTML. That's it!
